
# PantryPilot MVP Workspace

This folder keeps the Streamlit app and MVP-facing assets together.

## Current contents

- `app.py`: Streamlit entrypoint for the existing planner
- `data/raw/`: staging area for original local recipe files
- `data/processed/`: staging area for normalized recipe records

Milestone 1 adds dataset scaffolding only. The current planner still runs from the existing in-package sample dataset, so app behavior is unchanged.
